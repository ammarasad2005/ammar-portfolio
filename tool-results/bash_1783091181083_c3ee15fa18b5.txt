### Anti-patterns flagged this batch

- **Bare "Portfolio" title** (Vishal Bhangare) — reinforces B2 anti-pattern #21. Third corpus instance.
- **0 H2s and 0 H3s on home page** (Kavi Castelo, Vishal Bhangare) — reinforces B13 Aditya anti-pattern. Two more corpus instances. Most extreme heading-structure failures in the corpus alongside B13 Aditya.
- **0 meta tags** (Vishal Bhangare — no description, no keywords, no author, no OG, no twitter:card) — first corpus instance of zero meta tags. Page has no SEO surface at all.
- **H1 colored `rgb(197, 200, 201)` on pure white** (Vishal Bhangare) — first corpus instance of catastrophic H1 contrast failure. ~1.7:1 contrast ratio, well below WCAG AA 4.5:1. The page's primary heading is barely visible against the background.
- **`user-scalable=no` + `maximum-scale=2.5`** in viewport meta (Kavi Castelo) — first corpus instance of WCAG 1.4.4 violation (prevents pinch-zoom).
- **System-font-only H1** (Anurag Zete — `ui-sans-serif, system-ui, sans-serif, "Apple Color Emoji", "Segoe UI Emoji", "Segoe UI Symbol", "Noto Color Emoji"`) — reinforces B1 Krishnanand / B1 Bhushan anti-pattern. No typographic identity.
- **6-color Tailwind-default-blur-orb hero background** (Anurag Zete — blue/yellow/teal/pink/indigo/purple at 30% opacity) — most-trending 2024–2026 hero cliché. Reinforces B13 Dipak's "particle.js / lime-on-black AI-startup palette cliché" — the cliché is shifting from particle.js to multi-color-blur-orbs.
- **"Failed to fetch" rendered in DOM** (Anurag Zete — broken GitHub-stats widget without fallback) — reinforces B1 Bhushan "shipping broken third-party widgets" anti-pattern.
- **3 CSS frameworks simultaneously** (Kavi Castelo: Angular Material + Bootstrap 5 + Font Awesome kit) — first corpus instance at framework level. Reinforces B3 Gunjan's 64-tech-stack-PNGs anti-pattern at the framework level.
- **8 Google Fonts families loaded simultaneously** (Kavi Castelo: Roboto + Material Icons + Montserrat + Anton + Bebas Neue + Fira Code + Jacquard 12 + Freehand) — heaviest font payload in the corpus. Kitchen-sink font system.

- **14+ project cards without curation** (Vishal Bhangare — Live/Source link rows on every card) — reinforces B1 Bhushan "dumping every project without curation" anti-pattern.

### Adoptability recommendations for the premium portfolio being built

- **Adopt (strongly)**: Kavi Castelo's **event-driven sound as affordance feedback** — click sound on interactive elements + typing sound on text-reveal animations. This is the *correct* sound pattern (event-driven, not ambient). Avoid background music. Wire click sounds to primary CTAs (Resume download, project card clicks) and a typing sound to any typewriter-animation effect. Honor browser autoplay-with-sound policy (sound triggers on user interaction, not page load).
- **Adopt (strongly)**: Anurag Zete's **project status + date-range metadata** ("completed 9/11/2025 - 2/25/2026") on project cards. Refines B11/12 project tri-state status with *duration arithmetic*. Signals project lifecycle duration (months) — strong credibility signal if dates are accurate.
- **Adopt (strongly)**: Sambhu Babu's **strong security-header stack** (strict CSP + permissions-policy: `camera=(), microphone=(), geolocation=(), interest-cohort=()` + referrer-policy + X-Content-Type-Options + X-Frame-Options + X-XSS-Protection). Table-stakes for any modern portfolio. Most portfolios ship zero security headers.
